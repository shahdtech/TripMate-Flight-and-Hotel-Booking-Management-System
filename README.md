# TripMate-Flight-and-Hotel-Booking-Management-System

## Overview
TripMate is a comprehensive booking management system designed to handle flight and hotel reservations efficiently. The system provides users with a seamless experience for searching, booking, and managing travel services, while ensuring reliable data handling and integration between system components.

## Features
- Flight booking management  
- Hotel reservation system  
- User-friendly interface  
- Efficient data storage and retrieval  
- Integrated system components for smooth operation  
- Weather information integration for better travel planning  

## Technologies Used
- Programming Language: Java  
- Database: MySQL  
- Tools: NetBeans  

## System Design
- The user interfaces were developed using Java Swing (JFrame) for building graphical user interfaces.
- The system is connected to a local MySQL database to manage booking data.  
- Integrated with an external API to retrieve real-time weather information.  

## How to Run the Project

1. Set up the database:
   - make the SQL file into your MySQL database  

2. Run the application:
   - Open the project in NetBeans  
   - Run the welcome file  

3. Use the system:
   - Navigate through the interface to book flights and hotels
